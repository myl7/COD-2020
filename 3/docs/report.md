### DBU

增加 CPU 的引出端口，从而满足对寄存器堆和内存的 Debug 访问。

特别地，针对 CPU 的 `clk` 信号，使用了一个 Mux 来选择携带的时钟信号或是一个只翻转的按键控制信号，从而保证时钟信号的时序敏感。
